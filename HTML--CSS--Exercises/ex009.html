TRABALHANDO COM ID E CLASS NO CSS

ID no CSS

Não se deve utilizar o ID "#" como classe. Portanto, cada ID ("#") só deve ser atribuído a 1 elemento de HTML.

CLASS no CSS

A classe são seletores cuja função é atribuir estilo certo a um ou vários elementos de HTML.
