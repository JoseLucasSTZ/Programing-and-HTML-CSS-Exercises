Segurança no iFrame CUIDADO PARA QUEM VOCÊ VAI ABRIR AS PORTAS DO SEU SITE

Há utilização do iFrame traz algumas ameaças de segurança porque permite que um script de outro site seja executado no seu. Esse script pode conter vírus, ou ter seus formulários utilizados para phishing e outros golpes

Como forma de melhorar a segurança, é possivel utilizar os atributos "referrerpolicy" (no-referrer não permite capturar dados de ninguém) e "sandbox" para restringir os riscos. (exemplo abaixo)
