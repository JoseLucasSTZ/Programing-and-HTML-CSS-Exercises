O uso da tag <pre>

 
            
A Tag <pre> é importante porque garante uma pré-formatação do seu texto. Com ela se pode,
por exemplo, escrever um código diretamente no html com indentação, conforme se observa: 
        
            num = int(input("Digite um número"))
            if num == 10:
                print("Hello World!")
            else:
                print("Hello html!")

Uso de Citações

Como se sabe, a linguagem HTML é semântica, por isso é importante utilizar os marcadores corretos. Para as citações simples usa-se a tag <q> para inserir aspas duplas: Lorem ipsum, dolor sit amet consectetur adipisicing elit.

Por outro lado, para inserir uma citação completa, usa-se a tag <blockquote>:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Sunt earum, voluptatibus dolor blanditiis soluta laboriosam delectus non aliquid. Rem reiciendis cum ex reprehenderit corrupti deleniti nam tempore nihil iure magni!

É interessante utilizar o atributo CITE para referenciar a fonte.
